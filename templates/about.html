🎯 What is YouTube Downloader?

YouTube Downloader is a free, open-source web application that allows you to download videos and audio from YouTube in various qualities. Built with FastAPI and yt-dlp, it provides a fast, simple, and retro-styled interface for all your downloading needs.

✨ Features

🎬

Multiple Video Qualities

Download videos in 1080p, 720p, 480p, 360p, or 240p resolution

🎵

Audio Extraction

Extract audio from videos as high-quality MP3 files (192kbps)

📋

Playlist Support

Download entire playlists (up to {{ config.MAX_PLAYLIST_SIZE }} videos at once)

🔍

YouTube Search

Search YouTube directly and download without leaving the app

⚡

Fast & Lightweight

Built with FastAPI for blazing-fast performance and minimal resource usage

📱

Responsive Design

Works perfectly on desktop, tablet, and mobile devices

🔒

Privacy Focused

No registration required, no data collection, files auto-deleted

🛠️ Technology Stack

Backend

  • Python 3.x
  • FastAPI
  • yt-dlp
  • Uvicorn

Frontend

  • HTML5
  • CSS3
  • Vanilla JavaScript
  • Jinja2 Templates
